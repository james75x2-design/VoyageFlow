# WriCoRe Project Notes

## Purpose
WriCoRe is being upgraded from an AI workspace into a retrieval-grounded writing, coding, and research assistant.

## RAG Goal
The first RAG goal is to retrieve relevant knowledge-base chunks and return source-grounded answers with citations.

## Harness Goal
The harness should separate context, constraints, and evaluation so future changes can be tested instead of guessed.

## Current Scope
The first implementation should focus on ingestion, chunking, basic retrieval, and visible source IDs before adding vector databases or agent orchestration.
